# ARPG Prototype v0.1.0

## Windows Build

Download `arpg-prototype-win64.zip`, extract it, and run `arpg-prototype.exe`.

## Highlights

- Added a dedicated dev zone with ordered gathering nodes, stations, and service vendors.
- Expanded the item, recipe, and station data model for broader progression coverage.
- Added combat level groundwork, debug tooling, logs, tooltip support, and crafting timing.
- Improved UI structure, menu flow, and tester-facing Windows packaging.

## Known Notes

- This is still a development build.
- Saves and world cache are generated locally at runtime.
- First-pass balance and content coverage are in place, but tuning is ongoing.

## Controls

- `I`: Inventory
- `C`: Equipment
- `K`: Skills
- `L`: Game Log
- `J`: Combat Log
- `F4`: Combat Debugger
- `F1`: Developer Menu

## Installation

1. Download `arpg-prototype-win64.zip`
2. Extract to any writable folder
3. Run `arpg-prototype.exe`
